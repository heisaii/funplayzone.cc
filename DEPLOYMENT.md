# Deployment

This is a static site. It can be deployed to GitHub Pages, Cloudflare Pages, Netlify, Vercel, or any static host.

## GitHub Pages

- Build command: leave empty
- Publish directory: `/`
- Required runtime: none

## Content Flow

Recommended workflow:

1. Use GNews to discover music and World Cup topics.
2. Verify facts with FIFA, artist, label, venue, YouTube, Spotify, or event pages.
3. Generate an original article body.
4. Add source links to the Markdown article.
5. Add licensed images or stable remote image URLs.
6. Update `data/articles.js`.

Avoid copying full articles, lyrics, or unlicensed images.
